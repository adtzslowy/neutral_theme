Adtzslowy Nordish
Adtzslowy Nordish is a refined dark theme inspired by the Nord color palette, redesigned with a neutral foundation to provide a balanced and distraction-free coding environment.
It combines cool Nordic tones with a carefully tuned neutral background to deliver clarity, consistency, and visual comfort across modern development workflows.
Overview
This theme is built with a clear objective: reduce visual noise while preserving semantic clarity.
Instead of relying on strong contrast or aggressive color accents, Adtzslowy Nordish uses muted, harmonized tones to maintain focus during long coding sessions. The result is a calm, professional workspace suitable for full-stack development.
Design Principles
Neutral dark background to minimize eye strain
Cool gray foreground for comfortable readability
Muted Nord-inspired accents for structural clarity
No italic or bold styling to maintain visual consistency
Underline reserved exclusively for interactive elements
Consistent behavior across languages and frameworks
Every color choice is intentional and optimized for extended use.
Language Support
Adtzslowy Nordish is tested and tuned for:
HTML
CSS / SCSS / SASS
JavaScript
TypeScript
React (JSX / TSX)
Laravel (PHP & Blade)
Syntax highlighting remains consistent and balanced across these environments.
Key Highlights
Clear distinction between native HTML tags and React components
Soft but readable comment styling
Balanced keyword and function highlighting
Harmonized terminal ANSI colors
Low-stimulation design optimized for deep work
Recommended Settings
For the best experience:
Use a regular-weight monospaced font (e.g., JetBrains Mono, Fira Code, Cascadia Code)
Set monitor brightness to a moderate level
Enable custom title bar styling for full visual consistency
Installation
Install the .vsix package via:
code --install-extension adtzslowy-nordish-x.x.x.vsix
Open Command Palette
Select Color Theme
Choose Adtzslowy Nordish
